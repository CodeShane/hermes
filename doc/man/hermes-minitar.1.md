hermes-minitar(1) 
=============================

## SYNOPSIS

An archiver used internally by hermes, not intended to be used by end users,
but documented for completeness.

`hermes-namespace-container -cfptvxlz`

## DESCRIPTION

## OPTIONS

* -c:
  Create an archive.
* -f:
  Output file.
* -t:
  List an archive.
* -x:
  Extract an archive.
* -l:
  lz4 compression.
* -z:
  gzip compression.
